piFM-Improved
=============

Programme ajoutant plusieurs fonctionnalités à piFM :
* Gestion d'une playlist et des fonctionnalités associés (Lecture, pause, stop,...) ;
* Interface console (grâce à NCurses) ;
* Conversion des fichiers en parallèle.
